
Structure of folders
====================
Sudoku-C : main folder
    +- configurations : This folder contains the config.xml
    +- Lib : This folders contains all lib to use into the main program
    +- outputs : Folder by default out output files. (cvs, txt and html)
    +- unittests : This folder contains the unit test for the code
        +- sources : This folder contains the recourses to run the unit test
